# Big Basket EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Big Basket dataset using Python.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Outlier Detection using IQR
- Product Price Analysis
- Discount Analysis
- Data Visualization

## Visualizations
- Countplot
- Histogram
- Boxplot
- Scatterplot

## Key Insights
- Most products belong to a few major categories.
- Higher market prices generally have higher sale prices.
- Some outliers were detected in price-related columns.
- Missing values were handled successfully.
